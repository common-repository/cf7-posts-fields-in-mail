This plugin will help you to track the page, post, woocommerce product information where user will submitted the contact form you will get information in mail.

== Description ==
CF7 Posts Fields In Mail is an addon plugin for contact form 7 where you can get the post, page, product information like, page title, page url, post title, post url, product title, product url from where user the submitted the contact form 7.


= Features =
Following is a non comprehensive list of features:

* Enable/disable the plugin from option panel.
* Enable plugin settings for administrators.
* Enable/disable page contact form from administrators.
* Enable/disable post contact form from administrators.
* Enable/disable product contact form from administrators.

* CF7 Posts Fields In Mail is fully compatible with any types of themes and plugin but contact form 7 plugin is required for that.
We are adding new features as you are reading. So If your required feature is currently not available in this plugin, feel free to put up a request in the support forum for this plugin or revisit the page after a few weeks.

== Installation ==
1. Upload the `CF7 Posts Fields In Mail` folder to the `/wp-content/plugins/` directory OR search for "CF7 Posts Fields In Mail" from your WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a contact form using the contact form 7.
4. Put contact form 7 in you page post, product from admin or.
6. use the wordpress function do_shortcode in you theme files .

This plugin is created for Contact Form 7, so you must have to Contact Form 7 is installed in your WordPress website. If your WordPress website haven't Contact Form 7 installed then first install Contact Form 7 plugin. Please follow the link of WooCommerce plugin https://wordpress.org/plugins/contact-form-7/. If you are new in WordPress and don't know to how to install a WordPress plugin then follow the link https://codex.wordpress.org/Managing_Plugins. 
You can  find how to install WordPress plugin. For this plugin install you can also follow the same link.
CF7 Posts Fields In Mail maintain the standardization of wordpress.org . There are we are working on the standered hooks and filter for the given functionality.
